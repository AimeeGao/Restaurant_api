# 返回码定义

> 返回码采用 5 位  
> 以`999`（未知错误）为界  
> `< 999`为成功码  
> `> 999`为错误码

#### 特殊状态码

    999 未知错误
    0 ok

#### 1 开头为通用错误

    10000 通用参数错误
    10001 404 NotFound
    10004 403 禁止访问（权限不够）
    10005 授权失败（登录失败）
    10006 客户端类型参数错误

#### 2 开头为用户类型错误（包括验证码类）

#### 3 开头为课程类错误
